# 反机器人检测优化方案

## 问题背景

当前使用标准 Chrome DevTools MCP 模式操作小红书时，被平台检测为机器人并提示"违规行为，禁止发送内容"。

**检测原因分析：**
1. WebDriver 特征被暴露（navigator.webdriver = true）
2. 操作行为过于规律（固定延迟、直线鼠标运动）
3. 缺少真实浏览器的行为特征

## 方案设计

### 1. 浏览器指纹隐藏

#### 1.1 Chrome 启动参数
在 Chrome 启动时注入以下参数：

```javascript
const stealthArgs = [
  '--disable-blink-features=AutomationControlled',  // 禁用自动化检测
  '--disable-dev-shm-usage',                        // 避免共享内存检测
  '--no-sandbox',
  '--disable-setuid-sandbox',
  '--disable-infobars',                             // 隐藏自动化提示
  '--disable-browser-side-navigation',
  '--disable-web-security',                         // 禁用同源策略检测
  '--disable-features=VizDisplayCompositor',
  '--ignore-certificate-errors',
  '--disable-extensions',
  '--disable-plugins',
  '--disable-default-apps',
  '--disable-background-networking',
  '--disable-sync',
  '--metrics-recording-only',
  '--mute-audio',
  '--no-first-run',
  '--safebrowsing-disable-auto-update',
];
```

#### 1.2 CDP 脚本注入（页面加载后执行）

```javascript
// 去除 WebDriver 特征
Object.defineProperty(navigator, 'webdriver', {
  get: () => false,
  configurable: true
});

// 修改 plugins（让 Chrome 看起来有正常插件）
Object.defineProperty(navigator, 'plugins', {
  get: () => [1, 2, 3, 4, 5],
  configurable: true
});

// 修改 languages
Object.defineProperty(navigator, 'languages', {
  get: () => ['zh-CN', 'zh', 'en-US', 'en'],
  configurable: true
});

// 屏蔽自动化检测函数
window.chrome = {
  runtime: {}
};

// 重写 permissions 检测
const originalQuery = window.navigator.permissions.query;
window.navigator.permissions.query = (parameters) =>
  parameters.name === 'notifications'
    ? Promise.resolve({ state: Notification.permission })
    : originalQuery(parameters);

// 屏蔽 webdriver 检测
Object.defineProperty(navigator, 'webdriver', {
  get: () => undefined
});
```

### 2. 行为模拟系统

#### 2.1 随机延迟机制

```javascript
// 非线性随机延迟（人不会精确等待）
function humanDelay(min = 100, max = 2000) {
  // 使用beta分布，让延迟更接近某个峰值
  const u = Math.random();
  const beta = 2;
  const delay = min + (max - min) * Math.pow(u, beta);
  return Math.floor(delay);
}

// 操作间隔随机化
function getRandomInterval() {
  const min = 3000;  // 3秒
  const max = 15000; // 15秒
  return humanDelay(min, max);
}
```

#### 2.2 曲线鼠标移动

```javascript
// 贝塞尔曲线模拟人类鼠标轨迹
async function humanMoveMouse(page, targetX, targetY) {
  const startX = Math.random() * 100;
  const startY = Math.random() * 100;

  // 生成控制点（制造曲线）
  const controlPoints = [];
  const steps = 10 + Math.floor(Math.random() * 10); // 10-20步

  for (let i = 0; i <= steps; i++) {
    const t = i / steps;
    // 添加随机扰动，模拟人手抖
    const jitter = (Math.random() - 0.5) * 20;

    // 使用缓动函数（加速-匀速-减速）
    const ease = t < 0.3
      ? 2 * t * t                    // 加速
      : 1 - Math.pow(-2 * t + 2, 2) / 2; // 减速

    const x = startX + (targetX - startX) * ease + jitter;
    const y = startY + (targetY - startY) * ease + jitter;

    controlPoints.push({ x, y });
  }

  // 逐点移动
  for (const point of controlPoints) {
    await page.mouse.move(point.x, point.y);
    await page.waitForTimeout(10 + Math.random() * 20);
  }
}
```

#### 2.3 自然滚动模拟

```javascript
// 模拟人手滚动的非线性滚动
async function humanScroll(page, scrollY = null) {
  const viewportHeight = await page.evaluate(() => window.innerHeight);
  const scrollHeight = await page.evaluate(() => document.body.scrollHeight);

  // 随机滚动量（不是固定值）
  const scrollAmount = scrollY || (viewportHeight * (0.3 + Math.random() * 0.5));

  // 每次滚动使用平滑滚动
  await page.evaluate((amount) => {
    window.scrollBy({
      top: amount,
      behavior: 'smooth'
    });
  }, scrollAmount);

  // 滚动后随机停顿（模拟阅读）
  await page.waitForTimeout(500 + Math.random() * 2000);

  // 随机反向微小滚动（模拟调整阅读位置）
  if (Math.random() > 0.7) {
    await page.evaluate(() => {
      window.scrollBy({ top: -50 - Math.random() * 100, behavior: 'smooth' });
    });
    await page.waitForTimeout(300 + Math.random() * 500);
  }
}

// 随机滚动到页面不同位置
async function randomScroll(page) {
  const scrollHeight = await page.evaluate(() => document.body.scrollHeight);
  const viewportHeight = await page.evaluate(() => window.innerHeight);
  const maxScroll = scrollHeight - viewportHeight;

  if (maxScroll > 0) {
    const randomPosition = Math.random() * maxScroll;
    await page.evaluate((pos) => {
      window.scrollTo({ top: pos, behavior: 'smooth' });
    }, randomPosition);
    await page.waitForTimeout(1000 + Math.random() * 2000);
  }
}
```

#### 2.4 点击位置随机偏移

```javascript
// 点击时添加微小偏移，模拟点击不准
async function humanClick(page, selector) {
  const element = await page.$(selector);
  if (!element) return;

  const box = await element.boundingBox();
  if (!box) return;

  // 随机偏移（避免每次点击精确中心）
  const offsetX = (Math.random() - 0.5) * box.width * 0.6;
  const offsetY = (Math.random() - 0.5) * box.height * 0.6;

  // 先移动到元素附近
  await page.mouse.move(box.x + box.width / 2, box.y + box.height / 2);
  await page.waitForTimeout(100 + Math.random() * 300);

  // 点击
  await page.click(selector, { offset: { x: offsetX, y: offsetY } });
}

// 双击模拟
async function humanDoubleClick(page, selector) {
  await humanClick(page, selector);
  await page.waitForTimeout(50 + Math.random() * 100);
  await humanClick(page, selector);
}
```

#### 2.5 输入文本模拟（打字效果）

```javascript
// 模拟人类打字（随机间隔，不是匀速）
async function humanType(page, selector, text) {
  await page.click(selector);
  await page.waitForTimeout(200 + Math.random() * 300);

  for (const char of text) {
    await page.type(selector, char, {
      delay: 30 + Math.random() * 150  // 随机打字速度
    });

    // 偶尔停顿（模拟思考）
    if (Math.random() > 0.9) {
      await page.waitForTimeout(200 + Math.random() * 500);
    }
  }
}
```

### 4. 实现优先级

| 优先级 | 功能 | 效果 |
|--------|------|------|
| P0 | 隐藏 WebDriver 特征 | 解决基础检测 |
| P0 | 随机延迟 | 解决频率检测 |
| P1 | 曲线鼠标移动 | 解决行为分析 |
| P1 | 自然滚动 | 解决爬虫特征 |
| P2 | 打字模拟 | 提升输入真实性 |
| P2 | 点击偏移 | 避免固定模式 |

### 5. 风险提示

1. **过度优化风险**：行为太完美反而像机器人，要保留一定"不完美"
2. **性能影响**：模拟延迟会增加操作时间
3. **兼容性问题**：某些网站可能检测正常浏览器的特征

## 验收标准

- [ ] 隐藏 navigator.webdriver 属性
- [ ] 隐藏 Chrome DevTools 检测特征
- [ ] 鼠标移动使用曲线轨迹
- [ ] 操作间隔随机化（3-15秒）
- [ ] 滚动行为模拟人类
- [ ] 点击位置有随机偏移
- [ ] 打字输入有随机延迟
- [ ] 能够在小红书正常发布内容而不触发检测
