# 笔记发布实施计划

> **For Claude:** 使用 superpowers:subagent-driven-development 执行此计划

**目标：** 发布小红书笔记 - "恋爱中最大的谎言：他会给你安全感"

**内容定位：**
- 类型：情感态度类 - 极简扎心型
- 风格：扎心大实话，直接有力量

**配图需求：**
- 1张封面 + 1-2张配图
- 风格：极简文字风格 + 孤独感氛围图
- 比例：3:4（竖版图文最佳）

---

## 任务清单

### 任务1：生成配图

**工具选择（优先级）：**
1. baoyu-xhs-images（首选）
2. 即梦AI（备选）

**配图风格：**
- 内容类型：情绪管理/情感扎心类
- 推荐风格：warm（温暖扎心）或 minimal（极简）
- 布局：balanced 或 sparse

**具体操作：**
1. 使用 baoyu-xhs-images 生成 1张封面 + 1张配图
2. 或打开 https://jimeng.jianying.com/ai-tool/image/generate
3. 保存图片到 `aios/images/` 文件夹

**图片命名建议：**
- `2026-03-02-safety-cover.png`（封面）
- `2026-03-02-safety-1.png`（配图）

---

### 任务2：打开小红书创作平台

**URL：**
```
https://creator.xiaohongshu.com/publish/publish
```

**操作：**
1. 使用 browser_launch 启动浏览器
2. 使用 browser_navigate 导航到创作平台
3. 等待页面加载完成

---

### 任务3：上传图片

**操作：**
1. 点击上传图文按钮
2. 上传封面图和配图
3. 调整图片顺序（封面在第一张）

---

### 任务4：填写笔记内容

**标题：**
```
恋爱中最大的谎言：他会给你安全感
```

**正文：**
```
你为什么没有安全感？
因为你把所有幸福都押在别人身上。

他不回消息你就慌？
—— 因为你的世界只有他。

他去哪你要知道？
—— 因为你自己的生活太空了。

没有他你就活不了？
—— 因为你早已丢失了自己。

自己都给不了自己安全感，
凭什么指望别人？

把注意力收回到自己身上，
你若盛开，蝴蝶自来。
```

**话题标签：**
```
#恋爱 #情感 #恋爱观 #安全感 #人间清醒 #女性成长
```

3. 确认发布成功
